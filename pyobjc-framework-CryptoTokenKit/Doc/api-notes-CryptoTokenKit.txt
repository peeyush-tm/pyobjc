API notes: CryptoTokenKit framework
===================================

Apple Documentation
-------------------

The framework is documented on `Apple's developer website`__

.. __: http://developer.apple.com/library/mac/#documentation/CryptoTokenKit/Reference/CryptoTokenKitFrameworkRef/_index.html

API Notes
---------

The entire "CryptoTokenKit" framework on MacOSX 10.10 (or later) is available from Python.

.. note::

   The CryptoTokenKit framework is only available in 64-bit applications.
